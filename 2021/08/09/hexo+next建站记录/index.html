启动

说在开头, hexo是建站模板,而next是它的一款极致简约风格的黑白theme, 关于hexo和NexT的了解可以登录官网文档查看:

https://hexo.io/zh-cn/docs/

node.js就绪

首先下载node.js可以百度, 在此不多赘述

hexo安装

使用npm安装hexo-cli

npm install -g hexo-cli

npm是什么, JavaScript的npm, 就是python的pip, 是java的maven

启动

在/下开启git bash, 输入hexo server, 会在localhost:4000可视化你的博客, 适合提交前的本地调试

使用一句话发布所有更新的文章

hexo clean 清除缓存

hexo g / hexo generate, 从你的markdown文件生成博客的静态网页文件

hexo d / hexo depolyment, 向GitHub.io部署你的博客

更多的细节操作
