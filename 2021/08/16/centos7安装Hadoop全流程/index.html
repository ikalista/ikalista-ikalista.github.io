hadoop建立集群

Posted on 2021-08-16 Edited on 2021-08-23

1. 准备与规划

hadoop在本地建立集群进行测试

1.1. 硬件准备

虚拟机8G+内存, 50G+硬盘, CPU给到4个, 网络设置为host共享(只在本地建立集群进行测试)

1.2. 软件准备

下载centos7.iso, 教程自查

使用上一步下载下来的centos7.iso, 安装VMware Workstation 16 Player (注意是workstation, player版本的多开和复制会非常的麻烦) 教程自查

1.3. 主机规划

datanode 是 是
resourcemanager 是

zookeeper 是 是 是

2. 网络配置

2.1. 配置虚拟机网络

  • 建议使用静态路由(步骤不赘述, 我使用了第二种方法)
  • (可行, 不推荐) 在使用dhcp(动态路由协议)的情况下, 虚拟机会被自动分配到ip地址, 但是虚拟机网络中网络结构非常固定, 所以动态分布的ip地址其实约等于固定地址, 可以使用ifconfig将ip记录下来
